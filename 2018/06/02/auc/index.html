ROC,AUC评估软件或算法

生物信息软件实例评估讲解说明。可用auc对circRNA预测软件的评估,此类评估近期已有相关文献进行了报道，可作为进阶训练。

1.AUC说明

Auc通过将预测算法或软件的输出结果与已被验证的数据集对比进行评估。
已被证实的数据集包括正反两方面的结果，TPR与FPR反映预测算法正反两方面的错误率，通常两方面错误率都小的算法较优。

3.ROC曲线说明

预测值的阳性Label方法由阈值的选取决定，不同的阈值，阳性集合不同，方法如下：
选择第五列值进行计算,每次取其中一个数值为阈值（threshold),当第五列中的数值大于或等于这个阈值时,我们认为它为正样本,否则为负样本.记录该阈值下,正负样本个数,计算TPs,FPs,TNs,FNs,再计算TPR,FPR。

TPs：true positive数，
FPs：false positive数，
TNs：true negtive数，
FNs：false negtive数
TPR=TPs/Ps
FPR=FPs/Ns
Ps: 所有阳性个数
Ns: 所有阴性个数
真值 P P P N N N
预测值 P P N P N N
评价值 TP TP FN FP TN TN

TPR, 是阳性预测的准确率，即预测值评价为真阳性的个数（TPs）对所有阳性个数(Ps)的占比。
FPR, 是阴性预测的错误率，即预测值评价为假阳性（真值为阴性但预测值为阳性）的个数（FPs）对所有阴性个数(Ns)的占比。


以fpr为x，tpr为y，对每个阈值的(x,y)绘制散点图。

如：选择0.7523377414820508为阈值：正样本个数为2（0.7928078658665904，0.7523377414820508），负样本个数为2（0.7318917341670704，0.23268606897738542）。FN,TN,TP,FP分别是：1(0.7318917341670704)，1(0.23268606897738542)，2(0.7928078658665904，0.7523377414820508)，0。fpr,tpr分别是：0,0.66。
